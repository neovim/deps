Overview

MessagePack is an efficient binary serialization format.

It lets you exchange data among multiple languages like JSON but it's faster and smaller.

It's a pure Lua implementation, without dependency.

It's really fast with LuaJIT.

References

The MessagePack specification is available at github.com/msgpack/msgpack.

Status

lua-MessagePack is in production/stable stage.

It's developed for Lua 5.1, 5.2 & 5.3. There are 2 variants:

  • one compatible with all interpreters since Lua 5.1
  • another which uses the Lua 5.3 features

Download

lua-MessagePack source can be downloaded from GitHub.

Installation

lua-MessagePack is available via LuaRocks:

luarocks install lua-messagepack  # or lua-messagepack-lua53

or manually, with:

make install LUAVER=5.2

Test

The test suite requires the module lua-TestMore.

make test
